Processo Seletivo

O IFSP Hortolândia oferece diversas oportunidades de ingresso em seus cursos técnicos e superiores. Confira abaixo as principais informações sobre os processos seletivos:

Sisu

O Sistema de Seleção Unificada (Sisu) é utilizado para ingresso nos cursos superiores do IFSP. Os candidatos devem utilizar a nota do Enem para se inscrever. Confira os passos:

  1. Acesse o site oficial do Sisu durante o período de inscrições.
  2. Escolha o curso e o câmpus desejado.
  3. Acompanhe os resultados e, se aprovado, siga as instruções para matrícula.

Para mais informações, visite: sisu.mec.gov.br.

Vestibulares para Cursos Técnicos

Os cursos técnicos do IFSP possuem processos seletivos próprios. As inscrições geralmente ocorrem semestralmente. Confira os passos:

  1. Acesse o site oficial do IFSP Hortolândia.
  2. Leia o edital do processo seletivo.
  3. Preencha o formulário de inscrição online.
  4. Aguarde o resultado e siga as instruções para matrícula.

Para dúvidas, entre em contato pelo e-mail: vestibular@ifsp.edu.br.

Perguntas Frequentes

Veja as respostas para as dúvidas mais comuns:

  • Quem pode participar do Sisu? Qualquer pessoa que tenha realizado o Enem no ano anterior.
  • Os cursos técnicos possuem limite de idade? Não, os cursos técnicos são abertos para todas as idades.
  • Há taxa de inscrição? Não, os processos seletivos do IFSP são gratuitos.
  • Como saber mais sobre os cursos? Consulte a página de cursos no site oficial do IFSP Hortolândia.

Dicas para os Candidatos

Confira algumas dicas para se preparar para o processo seletivo:

  • Estude os conteúdos do Enem para o Sisu.
  • Leia atentamente o edital do processo seletivo.
  • Organize seus documentos com antecedência.
  • Participe de simulados e revise os conteúdos.
  • Entre em contato com o câmpus para esclarecer dúvidas.
